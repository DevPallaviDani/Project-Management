# 🚀 Project Management Dashboard

A modern **Project Management Dashboard** built with React that helps users manage projects, tasks, and track progress efficiently.

---------------------------------------------------------------------------------------------------------------------------------------

## 📌 Problem Statement

Managing multiple projects and tasks can become messy without proper organization. This application provides a simple and intuitive interface to:

* Create and manage projects
* Add and track tasks
* Monitor progress visually
* Stay organized and productive

--------------------------------------------------------------------------------------------------------------------------------------------------

## ✨ Features

* 📁 Create, edit, and delete projects
* ✅ Add, update, and manage tasks
* 📊 Track project progress
* 🎯 Task status management (To Do, In Progress, Done)
* ⚡ Quick action bar for faster workflow
* 🧩 Drag & Drop support (if implemented)
* 🌙 Dark mode UI (if implemented)

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **State Management:** React Hooks (useState / useReducer/custom hooks)
* **Drag & Drop:** @dnd-kit 
* **Build Tool:** Vite
